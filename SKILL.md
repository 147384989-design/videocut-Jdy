# 口播剪辑 v1.1

> Whisper 转录 + FFmpeg 静音检测 + 网页审核 + 自动剪辑 + 质量评估

## 功能

| 功能 | 说明 |
|------|------|
| **转录** | Whisper 离线转录，生成字级时间戳 |
| **Phrase 打包** | 将短句合并为完整语义单元 |
| **静音检测** | FFmpeg 检测 >0.5s 静音，自动标记 |
| **口误检测** | AI 分析重复句、语气词、重说纠正 |
| **时间线可视化** | HTML 波形图辅助审核 |
| **网页审核** | 可视化审核，点击执行剪辑 |
| **质量评估** | 剪辑后验证流畅度和压缩率 |
| **项目管理** | 保存/恢复编辑进度 |
| **高清导出** | 2-pass 编码，码率 1.2x |
| **字幕生成** | SRT/ASS 格式，支持烧录 |

## 使用方法

### 1. 剪口播（转录+审核+剪辑）

```
用户: /剪口播 视频.mp4
```

AI 会：
1. 提取音频
2. Whisper 转录（生成时间戳）
3. FFmpeg 检测静音
4. AI 分析口误
5. 启动审核服务器（http://localhost:8899）
6. 等待用户确认
7. 执行 FFmpeg 剪辑

---

### 2. 仅转录（不剪辑）

```
用户: /剪口播:转录 视频.mp4
```

只转录不剪辑，用于检查转录效果。

---

### 3. 生成字幕

```
用户: /剪口播:字幕 视频.mp4
```

生成 SRT 字幕文件。

---

### 4. 高清导出

```
用户: /剪口播:高清 视频.mp4
```

2-pass 编码，码率 1.2x 提升。

---

### 5. Phrase 打包 + 时间线

```
用户: /剪口播:时间线 视频.mp4
```

将 Whisper 短句合并为语义单元，生成 HTML 波形图辅助审核。
可用于可视化浏览转录结果。

---

### 6. 质量评估

```
用户: /剪口播:评估 输出目录
```

剪辑完成后评估：
- 流畅度（语气词/卡顿词残留）
- 压缩率（删除比例）
- 句子连贯性

输出 1-10 综合评分和建议。

---

### 7. 项目管理

```
用户: /剪口播:项目 <操作> <目录>
```

操作：
- `create` - 创建新项目，保存初始状态
- `confirm <start> <end>` - 确认删除片段
- `reject <问题ID>` - 拒绝删除，保留片段
- `complete` - 标记审核完成
- `export` - 导出最终删除列表

---

### 8. 自定义设置

```
用户: 静音阈值改成 1.5 秒
用户: 保留适量语气词
用户: 自动删重复句
```

AI 会记住偏好。

## 审核网页使用

1. 打开 http://localhost:8899
2. 播放视频，确认问题位置
3. 点击勾选要删除的项目
4. 点击「执行剪辑」
5. 等待完成后查看 output.mp4

### 快捷键

| 键 | 功能 |
|----|------|
| Space | 播放/暂停 |
| ← | 后退 5 秒 |
| → | 前进 5 秒 |
| Enter | 跳到选中项 |

### 批量操作

- 点击项目可选择/取消
- 「全选」/「全取消」批量操作
- 「导出列表」保存删除记录

## 常见问题

### Q: 转录很慢？

Whisper base 模型约 3 分钟转录 5 分钟视频（CPU）。如需更快，可改用 faster-whisper（需要 GPU）。

### Q: 审核网页打不开？

检查端口：`lsof -i :8899`（WSL）或 `netstat -ano | findstr 8899`（Windows）

### Q: 剪辑后音画不同步？

检查原视频编码，使用 `ffprobe video.mp4` 查看参数。

### Q: 如何跳过审核直接剪辑？

设置 `auto_delete_silence: true`（静音全自动），其他仍需确认。

## 配置

编辑 `config/settings.json` 调整默认参数：

```json
{
  "auto_delete": {
    "silence_over": 1.0,
    "auto_delete_silence": true,
    "confidence_threshold": 0.9
  },
  "thresholds": {
    "silence_min": 0.5,
    "repetition_chars": 5
  }
}
```
