# 卡顿词检测规则

## 定义

卡顿词是指说话过程中因思考、紧张或习惯导致的重复词汇，表现为同一词语连续重复多次。

## 常见卡顿词

| 类型 | 示例 |
|------|------|
| 短词重复 | "那个那个"、"就是就是"、"的话的话" |
| 代词重复 | "他他他"、"它它它"、"我我我" |
| 连词重复 | "然后然后"、"而且而且"、"所以所以" |
| 动词重复 | "关关掉"、"开开一下"、"看看看" |
| 混合型 | "这个这个这个"、"那个那个那个" |

## 检测规则

### 判断标准

1. **词重复** - 同一词语连续重复 ≥2 次
2. **粘连检测** - 重复词之间无其他词语
3. **边界识别** - 重复词与前后词语有明确边界

### 重复类型

| 类型 | 特征 | 示例 |
|------|------|------|
| 完全重复 | 字面完全相同 | "那个那个" |
| 部分重复 | 首字相同后续不同 | "看看看" |
| 混合重复 | 不同词交替重复 | "对对错对对错" |

## 处理方式

### 完全重复词
```
原: "那个那个今天讲"
处理: 删除前面的重复 → "今天讲"

原: "就是就是就是这个问题"
处理: 删除全部重复 → "这个问题"
```

### 部分重复词
```
原: "看看看这个图"
处理: 保留1个 → "看这个图"

原: "他他他不同意"
处理: 保留1个 → "他不同意"
```

### 被静音打断的重复
```
原: "关[静]关掉"
处理: 删除被截断的"关" → "关掉"
```

## 与语气词的区别

| 特征 | 卡顿词 | 语气词 |
|------|--------|--------|
| 本质 | 重复词汇 | 无意义填充 |
| 长度 | 固定词 | 单字或短词 |
| 可删除性 | 全部可删 | 部分需保留 |
| 语义影响 | 影响流畅度 | 影响自然度 |

## 连续卡顿检测

### 连续卡顿词
- 卡顿词后面紧跟另一个卡顿词 → 连续卡顿
- 处理：整个连续段视为一个整体处理

### 示例
```
原: "那个那个就是就是这个问题"
处理: 整段删除 → "这个问题"
```

## 删除优先级

| 优先级 | 情况 | 处理方式 |
|--------|------|----------|
| P0 | 独立卡顿词 | 直接删除 |
| P1 | 句首卡顿词 | 直接删除 |
| P2 | 句中卡顿词 | 删除+确认 |
| P3 | 句尾卡顿词 | 可保留 |

## 用户偏好

- `卡顿词全删` - 无条件删除所有卡顿词
- `保留句尾卡顿` - 句尾卡顿词可保留
- `遇到XX词直接删` - 指定特定卡顿词处理方式
