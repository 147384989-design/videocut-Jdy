# 用户习惯规则
# 这些规则用于AI分析时的决策参考

## 核心原则

**删前保后**：后说的通常更完整，所以：
- 删除前面的（口误/残句）
- 保留后面的（完整句）

**例外**：用户明确说"后面更顺"时，删前保后；用户明确说"前面更顺"时，删后保前。

## 静音段处理

| 静音长度 | 处理 |
|---------|------|
| ≤ 0.3秒 | 忽略 - 自然停顿 |
| 0.3-1秒 | 预选删除 - 句间卡顿 |
| > 1秒 | 建议删除 - 明显卡顿 |

特殊情况：
- 视频开头的静音必删
- 连续 5s+ 的静音整段标记

## 语气词检测

常见语气词：嗯、啊、呃、那个、就是、这个这个、的话、其实、然后、就是说

处理方式：标记但不自动删，除非连续出现多次。

## 重复句检测

判断方法：相邻句子开头≥5字相同

处理：删较短的句子（假设后者更完整）

## 卡顿词

"那个那个"、"就是就是"、"的话的话" 等重复词。

处理：删前面重复的部分。

## 重说纠正

模式：
1. 部分重复："你再关你关掉" → 删"你再关"
2. 否定纠正："它是它不是" → 删"它是"
3. 词被打断："依赖[静]依赖关系" → 删"依赖[静]"

## 残句检测

话说一半+静音 → 删整个残句

判断：句子明显不完整，且后面有静音。

## 自定义

用户可以通过对话告诉AI偏好，AI会记住：
- "静音阈值改成 1 秒"
- "保留适量嗯作为过渡"
- "遇到XX词直接删"
