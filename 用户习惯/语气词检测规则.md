# 语气词检测规则

## 常见语气词表

| 类别 | 语气词 |
|------|--------|
| 填充词 | 嗯、啊、呃、噢、哦、呀、哇、嘛、呢、啦、呐 |
| 犹豫词 | 这个、那个、就是、就是说、然后、其实、的话 |
| 连发词 | 这个这个、那个那个、就是就是、的话的话 |
| 口头禅 | 对吧、是吧、你知道吗、怎么说呢、怎么说来着 |

## 检测标准

### 单次出现
- 语气词单次出现 → 标记但不自动删
- 作为句子开头时 → 降权考虑

### 连续语气词
- 同一语气词连续出现 2 次 → 预选删除
- 连续出现 3 次以上 → 建议删除

### 位置权重
| 位置 | 权重调整 |
|------|----------|
| 句子开头 | 可接受度高 |
| 句子中间 | 可接受度低 |
| 句子结尾 | 可接受度高 |
| 独立成句 | 建议删除 |

## 处理方式

### 默认策略
1. 单次语气词 → 标记为"建议保留"，需用户确认
2. 连续语气词 → 标记为"建议删除"
3. 连续3次以上 → 自动删除

### 特殊情况
- 语气词后面是完整句子 → 可删
- 语气词后面是静音 → 删
- 语气词位于关键信息前 → 保留
- 用户明确偏好 → 按偏好处理

## 连续语气词示例

```
原: "嗯嗯嗯今天我们讲"
处理: 删除全部语气词 → "今天我们讲"

原: "那个那个说一下这个问题"
处理: 删除前两个 → "说一下这个问题"

原: "就是就是这个问题"
处理: 删除"就是就是" → "这个问题"
```

## 词频统计

分析时输出语气词统计：
- 各种语气词出现次数
- 连续语气词出现次数
- 语气词占总字数比例

## 用户偏好

用户可指定：
- `保留适量嗯作为过渡` - 保留少量语气词
- `遇到XX词直接删` - 指定特定词全部删除
- `语气词全删` - 无条件删除所有语气词
